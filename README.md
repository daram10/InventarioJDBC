# Inventario Supermercado - Java JDBC

Aplicación de consola en Java que gestiona el inventario de un supermercado de pequeña superficie mediante operaciones CRUD conectadas a una base de datos MySQL.

## Autora
Danna Ramirez
Programa: Análisis y Desarrollo de Software - SENA
Evidencia: GA7-220501096-AA2-EV01

## Tecnologías
- Java 17
- JDBC (conexión directa a MySQL)
- MySQL 8.0

## Estructura del proyecto
- `src/inventario/modelo/` → clase Producto
- `src/inventario/conexion/` → conexión a MySQL
- `src/inventario/operaciones/` → operaciones CRUD
- `src/inventario/main/` → clase principal
- `sql/` → script de base de datos

## Operaciones implementadas
- INSERT: agregar nuevo producto
- SELECT: listar todos los productos
- UPDATE: actualizar datos de un producto
- DELETE: eliminar un producto por ID
